## About
Tunez is a simple music bot that supports YouTube, Twitter, Newgrounds Audio, and Direct File Links.
## Commands
- play <url> | Plays the audio from that url/site
- skip | Skips the currently playing song
- loop <single, all, off> | Changes the loop mode
- stop | Stops all audio & leaves the voice channel
- queue |  Shows the queue
- help | Shows the help embed
